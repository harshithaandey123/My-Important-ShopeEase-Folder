# ShopEase Copilot

ShopEase Copilot is an AI-powered shopping assistant built using Python and Streamlit. It provides a conversational shopping experience where users can search products, receive recommendations, compare products, manage their cart, and interact using voice commands.

## Features

* Product Search
* Product Recommendations
* Product Comparison
* Cart Management
* Voice Search and Voice Responses
* Conversational Shopping Experience
* Session-Based Memory for Follow-up Actions

## Tech Stack

* Python
* Streamlit
* GitHub Copilot

## GitHub Copilot Usage

GitHub Copilot was used throughout the development process to:

* Accelerate code development
* Generate Streamlit UI components
* Assist with recommendation logic
* Support voice interaction implementation
* Improve code quality through refactoring suggestions
* Help with documentation and project structure

## Run Locally

```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```
